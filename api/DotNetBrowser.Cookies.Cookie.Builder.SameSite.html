SameSite

Gets or sets the SameSite attribute value of the cookie. This attribute denotes if your cookie should be restricted to a first-party or same-site context.

public SameSite SameSite { get; set; }

Property Value

SameSite

The SameSite attribute value.

Remarks

By default, the attribute value is set to LaxMode which means cookies are only set when the domain in the URL of the browser matches the domain of the cookie — a first-party cookie.

The None value requires that the Secure attribute is set to true.
