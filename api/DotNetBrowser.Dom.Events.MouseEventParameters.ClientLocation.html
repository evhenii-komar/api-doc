Property ClientLocation

ClientLocation

Gets or sets the location of the mouse cursor in the component's coordinate system at the time the event occurred.

Declaration
public Point ClientLocation { get; set; }
Property Value
Type Description
Point

the location of the mouse cursor in the component's coordinate system at the time the event occurred. For example, clicking in the top-left corner of the client area will always result in that the ClientLocation.X value equals 0, regardless of whether the page is scrolled horizontally.
